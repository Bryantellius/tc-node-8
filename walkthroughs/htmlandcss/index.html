HTML

HTML stands for Hyper-Text Markup Language and is used to give semantic meaning to text on a document.

Here is a look at the syntax rules:

  • Elements have names
  • Elements typically have opening and closings tags
  • An element example would be: <h1></h1>

HTML has two types of elements: block and inline.
